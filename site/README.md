# Tor VPN System - Website

This directory contains the GitHub Pages website for the Tor VPN System.

## Structure

- `index.html` - Main homepage
- `styles.css` - Futuristic styling
- `script.js` - Interactive features and animations

## Deployment

The site is configured to be deployed via GitHub Pages from the `site/` directory.

## Features

- Futuristic dark theme design
- Particle animations
- Interactive developer portal
- Documentation tabs
- Responsive design
- Smooth scroll animations
- Counter animations

## Customization

To customize the website, edit the following files:

- **index.html** - Content and structure
- **styles.css** - Colors, fonts, and styling
- **script.js** - Animations and interactivity
